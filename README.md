# Terraops dashboard app

## Tech Stack
- HTML/CSS
- Tailwind
- JavaScript (ES6)
- React + Vite
- react-apexcharts

### Installation:
1. Clone the Repository
    - git clone https://github.com/Vikas1798/Terraops_dashboard.git

2. Navigate to the Cloned Repository
    - cd your-repo-name

3. Install Dependencies
    - npm install

4. Start the Project
    - npm run dev
